# IT Controlling & Business Intelligence Portfolio

This repository demonstrates practical approaches to **IT controlling, financial reporting, and business intelligence** in an IT service environment.
The goal is to show how financial and operational IT data can be structured, analysed, and visualised to support **commercial decision making**.

The repository simulates a typical **IT controlling scenario** including service cost management, cost allocation, reporting, and automation concepts.

## Objectives

This project demonstrates:

* Structuring and standardising IT financial and operational data
* Creating datasets for **IT service cost analysis**
* Developing **commercial reporting structures**
* Preparing data models for **Business Intelligence dashboards**
* Demonstrating **cost allocation logic for IT services**
* Documenting automation concepts for IT controlling processes

## Example Use Cases

### 1. IT Service Cost Reporting

Analyse monthly costs for different IT services such as:

* Workplace services
* Infrastructure services
* Application services
* Network services

Example KPIs:

* Monthly service cost
* Cost per user
* Cost per device
* Plan vs Actual comparison

### 2. Cost Allocation

Simulation of how central IT costs can be distributed to business units using allocation drivers such as:

* Number of users
* Number of devices
* Infrastructure usage
* Service consumption

### 3. Business Intelligence Preparation

Data is prepared in a structure suitable for BI tools such as:

* Power BI
* QlikSense
* Excel reporting

The project includes examples of:

* Data modelling concepts
* Reporting views
* KPI definitions

## Repository Structure

```

SQL scripts for data modelling and reporting views

```
